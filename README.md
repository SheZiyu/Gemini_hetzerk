# DrugDiffusion

AI-powered structure-based drug design (SBDD) platform with 3D molecular visualization. This project provides an AlphaFold-style interface for visualizing drug candidates generated by diffusion models.

## Features

- **3D Molecular Visualization**: Interactive protein-ligand complex viewer powered by 3Dmol.js
- **Top-10 Drug Candidates**: Display ranked drug candidates with binding affinity scores
- **ADMET Properties**: Physicochemical properties and drug-likeness metrics
- **Lipinski Rule of 5**: Automatic validation of drug-like properties
- **AI Analysis**: LLM-generated explanations for each candidate
- **Multiple Render Styles**: Cartoon, lines, and surface visualization modes

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **UI Components**: shadcn/ui
- **3D Visualization**: 3Dmol.js
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

### Build for Production

```bash
npm run build
npm start
```

## Connecting to Backend

The mock service (`lib/mock-service.ts`) is designed to be easily replaced with real API calls:

```typescript
// Replace the mock function with real API
export const fetchDrugPrediction = async (pdbId: string): Promise<PredictionResponse> => {
  const response = await fetch('http://localhost:8000/api/generate', {
    method: 'POST',
    body: JSON.stringify({ pdb_id: pdbId }),
    headers: { 'Content-Type': 'application/json' }
  });

  if (!response.ok) throw new Error('Backend error');
  return response.json();
};
```
